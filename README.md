# Creepfolio Template 
by Paula Miguélez

You can find the deployed site here: [https://paulamiguelez.github.io/Plantilla_Portfolio/](https://paulamiguelez.github.io/Plantilla_Portfolio/)

Portfolio experimental inspirado en estéticas de horror digital,
interfaz retro y visual storytelling. Construido con HTML, CSS,
Bootstrap y GSAP.



## Concepto visual
El diseño toma referencias del horror digital, interfaces analógicas,
estética de creepypasta y UI retro. Se usan colores rojos, negros y blancos y
tipografías retro para reforzar la atmósfera.

## Página 404
Se incluye una página 404 personalizada con referencias a la estética
del proyecto.
